A shared document workspace for you and your AI agents.

Quarry is a real-time collaborative Markdown editor where agents are first-class collaborators — they join with their own presence and cursors, draft alongside you, leave comments, and propose suggestions you can accept or reject. Like a Google Doc, except agents connect over plain HTTP.

What you get

Real-time collaboration
Live cursors and presence for people and agents, in the same document at the same time.
Comments and suggestions
Threaded comments and tracked-change suggestions you accept or reject — from anyone in the doc.
Agent-native by design
A small REST API with presence, events, and atomic transactions. No SDK, no browser automation.
Share by URL
Every document is a private capability link. No accounts, no sign-in; anyone with the URL can join.
Markdown in, Markdown out
Every document is plain Markdown. Seed one from a file, edit it anywhere, and it round-trips cleanly to your repo, editor, or agent.
Ephemeral by default
Shared documents expire after 30 days unless you keep them, so scratch work stays scratch.
